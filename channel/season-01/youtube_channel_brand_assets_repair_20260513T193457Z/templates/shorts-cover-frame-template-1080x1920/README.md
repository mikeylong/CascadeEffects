# YouTube Shorts Episode Cover Frame Template

This is a reusable Shorts cover-frame template, not a separately uploaded Shorts thumbnail. Do not bake the Cascade of Effects channel title or channel logo into this template unless a specific episode package explicitly requests it.

Final cover frames must be generated later from the specific episode vertical source art, episode title/hook policy, and Shorts package manifest. Use `safe-zone-guide-overlay-1080x1920.png` only as a transparent guide layer.
