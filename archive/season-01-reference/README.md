# Season 1 Reference Archive

This directory is audit/reference material only. It preserves compact manifests,
checksums, receipts, selected production summaries, and legacy tests/scripts needed to
understand Season 1 without making Season 1 part of the active production surface.

Season 2 production must use `episodes/season-02/`, `ops/`, `apps/review/`, and the
packages under `packages/`. Large Season 1 media remains local under `.artifacts/season-01/`
and is intentionally ignored by Git.
