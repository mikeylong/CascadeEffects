# Therac-25 Living Cover Historical Notes

## Active Screen Correction

The operator console should not read as a dead or black screen. Historical interface references describe a computer terminal workflow for entering and editing treatment parameters, with the screen reporting treatment setup/status and sometimes cryptic malfunction codes. The source-art direction therefore uses an active period CRT/terminal with dim green or amber field rows, cursor/blocks, and treatment-parameter structure.

Readable generated screen text remains disallowed for this packet. Exact strings such as `MALFUNCTION 54`, dose values, mode names, or labels should not appear inside the ImageGen source art unless a later human-approved plan explicitly scopes deterministic text insertion.

## Script Alignment

The locked narration script uses `She` for the operator in the opening treatment-room sequence and again for the Tyler operator. The source-art packet therefore requires a female-presenting operator or a visually anonymous operator who does not contradict the script.
