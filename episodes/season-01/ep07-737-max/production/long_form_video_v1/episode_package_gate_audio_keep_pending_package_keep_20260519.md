## Core Mechanism

The changed condition is not simply a faulty angle-of-attack sensor. The MAX placed larger, repositioned engines on an older airframe family, creating a handling difference that threatened the commercial promise of 737 commonality. MCAS became the software answer to that contradiction.

The system failed to recognize that MCAS had grown from a background handling adjustment into a powerful repeated stabilizer command path tied to one sensor and mostly hidden from pilots. The blindness was institutional: commercial commonality, training minimization, delegated certification, and post-Lion Air procedural framing all reinforced the same assumption.

The consequence chain is structurally clear: commonality promise -> limited disclosure/training -> expanded MCAS authority -> single-sensor activation -> pilots fighting a failure mode they had not been fully told existed.
